## Principes de Design

### Core Principles

1. **Progressive Disclosure** : Commencer simple, révéler la complexité au besoin
2. **Immediate Feedback** : Chaque action doit avoir un retour visuel immédiat
3. **Undo/Redo** : Toutes les actions AI doivent être réversibles
4. **Transparency** : Montrer ce que l'AI fait et pourquoi
5. **Graceful Degradation** : Fallback manuel toujours disponible

### Action Cards

Quand l'AI effectue une action, elle affiche une carte de confirmation :

```tsx
interface ActionCardProps {
  action: AIAction;
  onUndo?: () => void;
}

function ActionCard({ action, onUndo }: ActionCardProps) {
  return (
    <div className="mt-2 rounded-lg border bg-card p-3">
      <div className="flex items-center justify-between">
        <div className="flex items-center gap-2">
          <CheckCircle className="h-4 w-4 text-green-500" />
          <span className="text-sm font-medium">
            {getActionLabel(action.type)}
          </span>
        </div>
        {onUndo && (
          <Button variant="ghost" size="sm" onClick={onUndo}>
            <Undo className="h-3 w-3 mr-1" />
            Annuler
          </Button>
        )}
      </div>
      {action.preview && (
        <div className="mt-2 flex gap-2">
          <img 
            src={action.preview.before} 
            alt="Before" 
            className="w-1/2 rounded border opacity-50" 
          />
          <img 
            src={action.preview.after} 
            alt="After" 
            className="w-1/2 rounded border" 
          />
        </div>
      )}
    </div>
  );
}

function getActionLabel(type: string): string {
  const labels: Record<string, string> = {
    'UPDATE_SECTION_PROPERTY': 'Propriété modifiée',
    'ADD_SECTION': 'Section ajoutée',
    'REMOVE_SECTION': 'Section supprimée',
    'REORDER_SECTIONS': 'Sections réorganisées',
    'CHANGE_VARIANT': 'Variante changée',
    'UPDATE_THEME': 'Thème modifié',
    'GENERATE_IMAGE': 'Image générée',
  };
  return labels[type] || 'Action effectuée';
}
```

### Implementation

```tsx
interface InlineAIMenuProps {
  selectedText: string;
  position: { x: number; y: number };
  onAction: (action: InlineAction, text: string) => Promise<void>;
  onClose: () => void;
}

function InlineAIMenu({ selectedText, position, onAction, onClose }: InlineAIMenuProps) {
  const [isLoading, setIsLoading] = useState(false);

  const handleAction = async (action: InlineAction) => {
    setIsLoading(true);
    try {
      await onAction(action, selectedText);
      onClose();
    } finally {
      setIsLoading(false);
    }
  };

  return (
    <div
      className="fixed z-50 rounded-lg border bg-popover shadow-lg"
      style={{ left: position.x, top: position.y }}
    >
      <div className="p-1">
        {INLINE_ACTIONS.map((action) => (
          <button
            key={action.id}
            onClick={() => handleAction(action)}
            disabled={isLoading}
            className="flex w-full items-center gap-2 rounded-md px-3 py-2 text-sm hover:bg-muted"
          >
            <span>{action.icon}</span>
            <span>{action.label}</span>
          </button>
        ))}
      </div>
    </div>
  );
}

const INLINE_ACTIONS = [
  { id: 'improve', icon: '🪄', label: 'Améliorer' },
  { id: 'rewrite', icon: '📝', label: 'Réécrire' },
  { id: 'translate', icon: '🌍', label: 'Traduire' },
  { id: 'shorten', icon: '✂️', label: 'Raccourcir' },
  { id: 'expand', icon: '📖', label: 'Développer' },
];
```

## States & Feedback

### Chat States

```tsx
type ChatState = 
  | 'idle'           // En attente d'input
  | 'sending'        // Message en cours d'envoi
  | 'streaming'      // Réception streaming
  | 'thinking'       // AI traite une action complexe
  | 'applying'       // Action en cours d'application
  | 'error';         // Erreur
```

### Visual Feedback

```tsx
function ChatStateIndicator({ state }: { state: ChatState }) {
  switch (state) {
    case 'idle':
      return null;
      
    case 'sending':
      return (
        <div className="flex items-center gap-2 text-muted-foreground">
          <Loader2 className="h-4 w-4 animate-spin" />
          <span className="text-sm">Envoi...</span>
        </div>
      );
      
    case 'streaming':
      return null; // Le texte qui apparaît est le feedback
      
    case 'thinking':
      return (
        <div className="flex items-center gap-1 text-indigo-500">
          <span className="animate-bounce delay-0">●</span>
          <span className="animate-bounce delay-100">●</span>
          <span className="animate-bounce delay-200">●</span>
        </div>
      );
      
    case 'applying':
      return (
        <div className="flex items-center gap-2 text-green-600">
          <Loader2 className="h-4 w-4 animate-spin" />
          <span className="text-sm">Application des modifications...</span>
        </div>
      );
      
    case 'error':
      return (
        <div className="flex items-center gap-2 text-destructive">
          <AlertCircle className="h-4 w-4" />
          <span className="text-sm">Une erreur est survenue</span>
        </div>
      );
  }
}
```

### Toast Notifications


## Accessibility

### Keyboard Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + J` | Ouvrir/fermer le chat AI |
| `Cmd/Ctrl + Enter` | Envoyer le message |
| `Escape` | Fermer le chat / Annuler |
| `Cmd/Ctrl + Z` | Annuler dernière action AI |
| `Cmd/Ctrl + Shift + Z` | Refaire dernière action |
| `Tab` | Naviguer entre quick actions |

### Screen Reader Support

```tsx
function ChatPanel() {
  return (
    <aside 
      role="complementary" 
      aria-label="Assistant AI ASAP"
      className="..."
    >
      <div role="log" aria-live="polite" aria-label="Messages">

```

## Métriques UX

### KPIs à Tracker

| Métrique | Description | Target |
|----------|-------------|--------|
| **Time to First Message** | Temps avant premier message envoyé | < 30s |
| **Messages per Session** | Engagement avec l'AI | > 5 |
| **Action Success Rate** | Actions réussies vs tentées | > 90% |
| **Undo Rate** | Fréquence d'annulation | < 10% |
| **Quick Action Usage** | Utilisation des raccourcis | > 30% |
| **Panel Open Rate** | % sessions avec chat ouvert | > 60% |

### User Research Signals

```typescript
// Events à tracker pour UX research
trackEvent('ai_panel_opened', { source: 'keyboard' | 'button' });
trackEvent('ai_message_sent', { message_length, has_quick_action });
trackEvent('ai_action_applied', { action_type, time_to_apply });
trackEvent('ai_action_undone', { action_type, time_since_apply });
trackEvent('ai_suggestion_shown', { suggestion_type });

trackEvent('ai_suggestion_dismissed', { suggestion_type });
trackEvent('ai_inline_menu_opened', { selected_text_length });
trackEvent('ai_inline_action_applied', { action_type });
```
